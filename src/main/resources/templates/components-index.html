About This Component Library

This component library is built with modern web technologies:

  • Thymeleaf - Server-side templating with fragment support
  • Tailwind CSS v4 - Utility-first CSS framework
  • Alpine.js v3 - Lightweight JavaScript framework for interactivity
  • Chart.js - Beautiful charts and data visualization
